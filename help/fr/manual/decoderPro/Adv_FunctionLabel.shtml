Ce volet est plus qu'un bloc-notes pour garder une trace de ce qui arrive quand les boutons de fonction sont appuyés.  Il interagit aussi avec le Régulateur JMRI.

L'Onglet Étiquette de Fonction fait les réglages pour le Logiciel Régulateurs au sein de JMRI. Principalement les Régulateurs à l'écran, mais certains paramètres sont en miroir sur les téléphones intelligents ( iPhone, Android ) utlisant l'interface WiThrottle.

Quand vous utiliser la liste déroulante sur le régulateur JMRI pour sélectionner une loco depuis votre inventaire, les boutons fonctions seront marqués avec les descriptions que vous avez entrées pour ces;fonctions dans ce volet.

Dans cet exemple, les touches Fonctions sur le Régulateur sont libellées pour une FP-45 avec un décodeur Son Tsunami avec les fonctions sons et éclairage. .

L'usage des cases, les images, la définition du type de bouton et le bouton radio à droite de chaque fonction peuvent être définis:

verrou

Rends le bouton de fonction "verrouillable" ou "non-verrouillable". Quand la case est cochée la fonction est "verrouillée"

Fonction Verrouillée Quand présseé restera On(position basse) jusqu'à ce que vous pressiez encore le bouton ex: bouton Fonction libelé "Cloche" tintera jusqu'à ce que vous pressiez encore.

Fonction Non-Verouillée quand vous pressez la fonction bascule On etpuis Off. ex: bouton Fonction libélé "corne brève", quand pressé fournit un coup de corne.
off Image pour l'icône qui est affichée sur le Bouton Fonction quand il est à l'état Off.
on Image pour l'icône qui est affichée sur le Bouton Fonction quand il est à l'état On.
shunt Définit une touche de fonction "virtuelle" pour le logiciel JMRI. L'option "verrou" nécessite aussi d'être définie simultanément avec l'activation de la touche de fonction virtuelle "shunt". Quand cette touche de Fonction est pressée le curseur de vitesse du régulateur sur le logiciel régulateur change pour le style zéro central, plutôt que la "course complète avec la touche de direction séparée"

You may drag and drop icons into the off or on space holder herer or in the Throttle Function pane properties.

You may also edit the function button text in the JMRI Throttle and save those changes to the selected Roster entry. To use these changed entries in the future, make sure you click on the to save these values into the Roster entry that is in use.
