All of the JMRI functions - including the OperationsPro Program - can be accessed via the menu bar on either the DecoderPro® or PanelPro main window. Additional menus are added when functions are selected, see the example below with added Operations menu (selected option on the OperationsPro Settings pane). All OperationsPro functionality is also in the Tools menu on the main JMRI toolbar.

DecoderPro® icon and DecoderPro® version information is displayed. The version of Java installed and in use is also shown.

Menu Bar

Contains the menus that allow you to access all the features and tools in JMRI.

This Manual focusses on the functions and menus that apply to JMRI® OperationsPro. Please consult the other JMRI® Manuals for other functions.
