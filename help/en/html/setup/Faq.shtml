Why does ant stop with a getX() error? (advanced)
When you try to build JMRI using an IDE or ant and a build fails with an error like "method getX() not found", this may indicate that an older version of a specific java library installed on your system prevents ant from finding the proper parts. Ant uses the JDK (Java Development Kit) and additional Java modules supplied with the JMRI source code to build parts of JMRI.
To diagnose a possible issue causing the above error in OS X, open Terminal and type find /System/Library/Java -name \*.jar ! -name jdk\* -exec pkgutil --file-info {} \;
