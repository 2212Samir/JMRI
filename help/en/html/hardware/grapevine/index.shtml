Hardware Support: Grapevine System

Naming and input/output connectors are described on a separate page.

Tools

You can check whether your Grapevine is working and configure the connected nodes using the Node Table tool. It also lets you change the number of a node.

Limitations

  • Fast clocks not yet supported. If you have one and want to test it with JMRI, let us know.
